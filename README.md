## Setup

To run this project, install it locally using npm:

```
$ git clone https://github.com/baonguyen2411/vite-react-tailwind.git .
$ cd vite-react-tailwind
$ npm install
$ npm run dev
```

To build this project:

```
$ npm run build
```
